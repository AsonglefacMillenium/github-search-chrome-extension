# GitHub Search for Chrome

Search GitHub repositories from your omnibar.

## How?

Type **gh** and **Tab** and find some repos.

### Supported queries.

- ```searchterm``` Searches all repos.
- ```username/searchterm``` Searches the repos of a user.
